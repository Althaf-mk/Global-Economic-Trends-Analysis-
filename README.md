
# 🌍 Global Economic Trends Analysis using IMF World Economic Outlook (WEO) Dataset

## 📌 Project Overview

This project analyzes global economic trends using the International Monetary Fund (IMF) World Economic Outlook (WEO) dataset. The objective is to clean, process, visualize, and extract meaningful insights from economic indicators across multiple countries and years.

The project demonstrates practical applications of Data Analytics using Python libraries such as Pandas, NumPy, Matplotlib, and Plotly.

---

## 🎯 Objectives

- Perform data cleaning and preprocessing
- Handle missing values and duplicates
- Explore global economic trends
- Analyze country-wise economic performance
- Create interactive and static visualizations
- Generate insights from economic indicators
